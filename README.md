# FastAPI Weather Analysis API

A weather API built with FastAPI that provides day-to-day weather analysis
(min/max/avg temperature, humidity, wind, pressure, cloud, rain and pop).

## How to run

1. Clone repo  
2. Create virtual env  
3. Install dependencies  
4. Run API with:

uvicorn main:app --reload

## Endpoints

GET /daily-analysis?city=Hyderabad
